DOM-querying fragment

document.documentElement assertions

document.documentElement.tagName === "HTML"

document.querySelector('html') returns document.documentElement element

document.querySelector('wf-html') returns document.documentElement element

document.body assertions

document.body.tagName === "BODY"

document.querySelector('body') returns document.body element

document.querySelector('wf-body') returns document.body element

document.head assertions

document.head.tagName === "HEAD"

document.querySelector('head') returns document.head element

document.querySelector('wf-head') returns document.head element

Selectors

Child selector: document.querySelector('body > h2') returns h2 element

Descendant selector: document.querySelector('body h2') returns h2 element

Next sibling selector: document.querySelector('body > h2 + section') returns section

Class selector: document.querySelector('.body') returns body section (*not document.body)

ID selector: document.querySelector('#body') returns body section (*not document.body)
